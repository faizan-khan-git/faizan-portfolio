# Md Faizan Khan Portfolio

React + Vite portfolio built from the CV in this folder.

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown by Vite, usually:

```text
http://127.0.0.1:5173/
```

Do not open `index.html` directly with Live Server for development. This project uses React JSX, so it must run through Vite.

## Build For Deployment

```bash
npm run build
```

Deploy the `dist` folder. The Vite config uses relative asset paths, so it works on subfolder hosting such as GitHub Pages.
